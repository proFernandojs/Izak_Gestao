## 🛠️ TROUBLESHOOTING

### ❌ "URL não apareceu"

```
1. Feche iniciar-ngrok.bat
2. Certifique-se que servidor está rodando (node server.js)
3. Abra iniciar-ngrok.bat novamente
4. Aguarde 5 segundos
```

### ❌ "Não consigo acessar a URL"

```
1. Verifique se Ngrok está rodando (janela aberta)
2. Copie corretamente a URL
3. Confirme que tem internet/dados móveis
4. Tente em outro navegador
```

### ❌ "URL mudou"

```
Toda vez que reinicia Ngrok, a URL muda (comportamento normal)
Nova URL é gerada automaticamente
```
